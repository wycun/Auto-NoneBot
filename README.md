# Auto-NoneBot
自动部署 Nonebot / QQ协议 / Python(3.12.0)

# 程序下载 (Linux/Windows 系统)
```python

git clone https://github.com/wycun/Auto-NoneBot.git && cd Auto-NoneBot

```

# 运行程序 (Linux 系统)
```python

sudo chmod +x 启动.sh && sudo ./启动.sh

```

# 运行程序 (Windows 系统)
```python

Start-Process -FilePath ".\启动.bat" -Verb RunAs

```
